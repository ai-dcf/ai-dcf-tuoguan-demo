# 托管班 H5 需求澄清 Demo

这是一个基于 **托管班设计需求-v3.md** 快速构建的纯静态 H5 Demo，旨在辅助与客户澄清核心业务流程与交互细节。

## 🎯 已实现核心场景

本项目重点还原了 PRD 中变更较大及交互复杂的模块：

1.  **5.3 作业发布 (HomeworkView)**
    *   **核心变更**：仅保留作业发布与管理（班级维度），**移除**了学生提交列表（按 V3 需求）。
    *   **交互**：支持新建作业、发布/草稿状态切换、学科筛选。
    *   **验证点**：确认“只管发不管收”的流程是否符合预期。

2.  **5.4 作业点评 (ReviewView)**
    *   **核心变更**：集成了作业批改、错题录入、表现点评的闭环流程。
    *   **交互**：
        *   **错题联动**：点击“错题”按钮可模拟录入错题数量，并在作业卡片上实时显示。
        *   **状态切换**：一键切换未交/已交/补交状态。
        *   **快捷评级**：A/B/C/D 快速打分。
        *   **表现点评**：一键选择标签（专注、积极等）。
    *   **验证点**：确认“边改作业边录错题”的单屏操作效率。

3.  **5.2 考勤点名 (AttendanceView)**
    *   **交互**：支持批量签到/签退，以及单个学生的请假/状态变更。
    *   **验证点**：确认点名操作的便捷性。

4.  **4.1 首页总览 (HomePage)**
    *   **功能**：展示快捷功能入口与托管班级列表，作为进入各班级详情的入口。

## 🚀 如何运行

本项目已配置好所有依赖，您可以直接运行：

### 1. 启动本地预览 (推荐)
```bash
npm run dev
```
启动后打开浏览器访问控制台输出的地址 (通常是 http://localhost:5173)，即可在手机模式下体验交互。

### 2. 构建静态文件
```bash
npm run build
```
构建产物位于 `dist/` 目录，可直接部署到任何静态服务器或发送给客户体验。

## 🛠️ 技术栈
*   **React 19 + TypeScript**: 现代前端框架
*   **Tailwind CSS v4**: 高性能样式引擎 (已配置 Vite 插件)
*   **Lucide React**: 统一图标库
*   **Vite**: 极速构建工具

## ⚠️ 注意事项
*   本项目为**纯静态 Demo**，所有数据均为本地 Mock 数据，刷新页面后会重置。
*   部分次要页面（如历史记录详情、设置深层页）仅作为占位展示，不包含完整逻辑。
